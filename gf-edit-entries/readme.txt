== Description ==

Do you need to edit any of the Gravity Forms entries on your site? 

[Bright Vessel](https://www.brightvessel.com/) was tasked to allow edits for Gravity Forms to make it easier for clients to edit and we wanted to share with the community.

Here is a [PDF user’s guide](https://share.brightvessel.com/MA1).


We have been featured as an [official extension](https://share.brightvessel.com/MAz) on gravity forms website.



Have any ideas on how we can improve the plugin? Don’t hesitate to share it [here](https://share.brightvessel.com/MAg) and we will try to implement it in future versions of the plugin. 

Do you like the plugin and want to say about it?

[Write a review and rate it here.](https://share.brightvessel.com/MAe)

Are you interested in other our WordPress plugins?

[Block IPS for Gravity Forms](https://share.brightvessel.com/iYG) - Block IP addresses from submitting forms.
[Preorders for WooCommerce](https://share.brightvessel.com/MAA) - Allow customers to preorder products.


== Installation ==

1. Upload the folder `gf-edit-entries` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
